# RG3 홍삼 마케팅 전략 프로젝트

RG3 홍삼의 효과적인 마케팅 전략을 기획하고 정리하는 프로젝트입니다.

## 시작하기

1. `index.html` 파일을 브라우저에서 열면 바로 확인할 수 있습니다.
2. 별도의 설치 과정이 필요 없습니다.

## 폴더 구조

```
rg3-hongsam-marketing/
├── css/
│   └── style.css       # 스타일(디자인) 파일
├── js/
│   └── main.js         # 자바스크립트(동작) 파일
├── images/             # 이미지 파일 폴더
├── index.html          # 메인 웹페이지
├── .gitignore          # Git 제외 파일 목록
└── README.md           # 프로젝트 설명서
```
